Connect your Google account to read and create events. How to get a Client ID ›

  1. Go to Cloud Console → create a project.
  2. Enable the Google Calendar API (and Tasks API).
  3. OAuth consent screen → External → add your email as a test user. Then set publishing status to In production (so logins don't expire weekly).
  4. Credentials → Create credentials → OAuth client ID → Desktop app.
  5. Copy the Client ID and Client secret below.
